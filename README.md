# KreativKid Portfolio

KreativKid Portfolio is a modern personal portfolio website built to showcase projects, skills, and creative work in a clean and minimal interface. The goal of the project is to present development work, experiments, and ideas in a visually appealing and organized way.

## Features

Modern Minimal UI
Clean layout with a focus on projects and content.

Project Showcase
Displays selected projects with descriptions, tech stack, and links.

Responsive Design
Works smoothly on desktop, tablet, and mobile devices.

Fast Performance
Optimized layout and lightweight components for quick loading.

Simple Navigation
Easy navigation between sections like Home, Projects, and Contact.

## Tech Stack

Frontend
React
Tailwind CSS

Other Tools
JavaScript
Modern UI design principles

## Installation

Clone the repository

```
git clone https://github.com/yourusername/kreativkid-portfolio.git
cd kreativkid-portfolio
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open in browser

```
http://localhost:3000
```
